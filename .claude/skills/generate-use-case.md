# Skill: generate-use-case

## Cuándo se usa
Cuando hay que documentar un caso de uso formal para el proyecto.

## Pasos que siempre seguís
1. Nombre del caso de uso
2. Actor principal
3. Precondiciones
4. Flujo principal paso a paso
5. Flujos alternos (qué pasa si algo falla)
6. Postcondiciones

## Formato de salida
Documento formal en español, estilo académico,
listo para incluir en el informe del Proceso Unificado.
